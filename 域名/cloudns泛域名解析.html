# 泛域名解析含义

泛域名解析是指将需要双向解析的域名进行泛化（又指通用化）

将原本需要两端都需要配置（双向解析）的 DNS 尽量简化为单向解析

另外双向解析域名还可以进行优选 IP or 优选域名

# Cloudflare 泛解析双向域名

首先将 cloudns 域名托管到 cloudflare

将 cloudflare 名称服务器指向到 cloudns 的 ns记录 （两条）

将 cloudflare 边缘证书指向到 cloudns 的 ns记录 （两条）

# 方式一：优选 IP

cloudns 指定 A记录 * 到 优选IP

cloudflare 指定 A记录 * 到 8.8.8.8 （用于指向云服务器，如果没有则可以不指定）

# 方式二：优选域名

cloudns 指定 CNAME记录 * 到 优选域名

cloudflare 指定 A记录 * 到 8.8.8.8 （用于指向云服务器，如果没有则可以不指定）
