# 下载内核

下载 Frp 内核

下载最新版内核即可

# 配置

以网页上给出的配置为准

选择内核，填写 服务器地址 ， 服务器端口 ， 验证方式=token ， 令牌=token

新建一个穿透服务

代理类型： tcp

代理名称： 生成

内网地址： 127.0.0.1

内网端口： 443

外网端口：端口需要在 Frp页面 给出的 端口允许范围 之内（常用端口基本占用，比如 80,443）

# 报错解决方案

点击启动，查看日志

点击右上角的刷新，出现 start proxy success 代表成功穿透

start error:port ybavailable 端口已占用

connection host... ，主机连接错误，请检查 服务器地址 ， 端口 ， token
