# 为什么要重装系统？

  1. 当电脑处于不可开机的状态
  2. 系统被黑客入侵，对操作系统进行了破坏
  3. 用户将操作系统损坏，无法开机
  4. 电脑卡顿（排除电脑本身配置差，这个只能换）

重启能解决 80% 的问题，重装能解决 99% 的问题，剩下的 1% 需要花钱

学会如何重装系统，也就相当于学会了一门稳赚 50 块钱的方法（修电脑的就是 50 块钱重装一次，别问我怎么知道的，不要花冤枉钱）

# 查看自己的 windows 版本

  • 设置 ——> 系统 ——> 关于 ——>windows 规格 ——> 版本

  • WIN+R，输入 winver ，查看当前 windows 版本

Windows11 目前的版本号有 21H2，22H2，23H2，24H2

最新版本号为 24H2 ，查看官网信息

Windows10 目前的版本号有 21H2 ，22H2（Windows10 将在 2025 年 10 月 14 日停止更新），查看官网信息

# 重装条件

要重装系统，需要一个 U 盘存当介质，一般推荐入手固态移动硬盘

# PE 系统

PE 系统是 windows 推出的一种维护系统，专门用户修复 windows 的微型系统，早期 PE 系统只能存在于软盘中（过时），现在可以存在于 U 盘，提供了很多便捷

# 重装步骤

# ISO 文件作为升级程序

当一台正常的 windows 电脑需要升级时，直接进行挂载，然后打开 setup.exe 进行升级即可

检查完成后，需要下载对应的文件，安装后即可升级完成

# ISO 文件作为重装程序

首先，如果你的设备上如果有重要的文件，请先备份！！！

# ISO 文件自动配置分区并安装

注意，分区后，磁盘中的所有数据将会被删除（格式化）谨慎操作！

打开 分区工具DiskGenius

选择你的硬盘 ——> 删除所有分区 ——> 保存

找到 ISO 镜像文件 ——> 双击 or 装载 ——> 点击 setup.exe

前置配置

选择安装 Windows11 ，同意协议

暂时选择没有产品密钥

选择 Windows11专业工作站版

选择安装磁盘，显示为 未分配空间 ， 70.0GB

安装

在重启后，如果出现不能正常引导启动的情况，就需要修复引导项（解决方案在下面）

正常进入系统前置设置界面

等待检查完成

设置网络连接，这里不要进行联网安装，很耗时间

  1. 按 shift+F10 打开命令提示符

  2. 输入 start ms-cxh:localonly

  3. 在接下来的窗口输入用户名和密码

创建用户

安装等待

以用户权限安装成功

# 重装时会需要的一些问题

# 这台电脑当前不满足 windows11 系统要求

这种情况是旧设备不支持安装 Windows11（不建议安装，如果硬要安装，可以参照以下做法）

# 解决方案

原因：大部分情况是因为旧电脑上没有 TPM2.0 ， TMP2.0 ：可信平台模块

可以有两种解决方案

1. 使用 easyRc一键安装 直接安装

2. 使用 Rufus 写盘工具，将 ISO 文件写入 U 盘中，写入 U 盘时可以取消该限制

# 系统引导出现问题，不能正常启动

# 解决方案

如果启动分区被删除了则需要重新安装

如果没有删除，则打开 Dism++ ——> 选择系统 ——> 打开会话

工具箱 ——> 引导修复

# ISO 附带的捆绑软件

非官方渠道的，大部分存在广告的 ISO，安装后会有各种捆绑软件，比如 360，鲁大师

推荐三个下载 ISO 镜像渠道

# 驱动安装

一般重装后，连接网络会自动下载驱动，如果没有正常更新驱动，需要手动通过驱动软件进行驱动更新

Q：为什么要更新驱动？

T：因为操作系统和硬件本身不能直接进行使用，需要安装驱动才能正常使用，一般出现蓝屏，可能是驱动出现了问题，解决方案直接更新稳定版的驱动即可

下载链接：https://www.123912.com/s/PlHbVv-XBwb（单文件版本，不需要安装，直接使用即可）
