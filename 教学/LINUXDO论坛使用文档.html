# 什么是 LINUXDO

LINUXDO 是 2024 年年初新启的一种社区性论坛

Q：为什么会出现这个论坛？

T：算是时代的产物？时候到了就出现了（论坛主：这是一项社会性调查实验）

Q：我为什么推荐这个论坛？

T：能让你认识这个世界更多，拉近信息差

# 关于账号

这个账号是在论坛开放注册的时候，我抽空注册的，现在是审核注册（写小作文给论坛主【50 字以上，非 AI】才可以获得一个账号），账号可以在闲鱼里搜，现在很多都卖 50+（三级账号），勉强算值钱吧

# 账号等级

每个账号都有属于自己的账号等级，等级越高，能查看的东西就越多

# 如何升级？

简而言之，在社区活跃就行了

访问次数：每日查看一次帖子可 + 1

回复话题：给别人回复话题

浏览的话题：查看别人的帖子

已读帖子：查看别人的回复

被举报的帖子：在创建帖子和回复别人的时候被别人举报，会 + 1，什么时候会被举报，具体看疑难解答

点赞：给别人的 帖子或 回复 点赞

获赞：别人给你点赞

被禁言：一般很少，除非在频道（实时聊天）中发签到（严禁）

# 社区功能

每个分类中又分等级限制，最高到 lv3，等账号到 3 级时候就都能看到了（3 级的好东西确实多）

差不多就这些要注意的

至于有多少好东西，就看我发的那个帖子，收获会很多，要看一些信息差就看前沿快讯，基本上都是很快的新闻，更多的是关于 AI 的新闻，其他的一些新闻也会有
