# Google

Google 是全球名列前茅的搜索引擎，可以用于学术搜索，也可以用于授权登录，比国内的百度更好用

# Google 账号

Google 账号用于登录 Google 邮箱

# 注册

# 电脑上进行注册（不推荐）

注册须知：需要科学上网（或者叫梯子）环境

注册 Google 账号不一定非要 Google 浏览器才能注册，其他浏览器也可以，只需要进入 Google 主页，进入登录页面即可

点击登录

点击创建账号 ——> 个人用途

随意填写姓名和姓氏

出生年月尽量超过 18 岁即可

选择你的邮箱前缀，或者自定义你的邮箱前缀

验证手机号（如果你的 IP 纯净度非常高，也许就不用验证你是不是自动程序）

注册成功

# 安卓手机上进行注册（推荐）

开启科学上网后，打开设置 ——> 进入 Google

进入 Google 服务，点击登录

点击创建账号 ——> 个人用途

随意填写姓氏和名称，出生年月满 18 岁，选择自己喜欢的邮箱前缀，输入密码，

出现这个界面就可以直接下一步后直接登录了

注册成功

# 账号保活步骤

尽量开启 2FA（二步验证）

点击账号 ——> 管理您的 Google 账号 ——> 安全性 ——> 二步验证 ——> 选择身份验证器 ——> 设置身份验证器

打开微软身份验证器 ——> 登录微软账号 ——> 点下面的二维码进行扫描 ——> 在手机上输入对应的 6 位数字进行验证 ——> 开启二步验证 ——> 继续 ——> 开启二步验证

# Google 邮箱

拥有 Google 邮箱后，Gmail 邮箱和 QQ 邮箱一样，可以收发邮件

还能用于登录 ChatGPT，Google AI Studio，Grok 以及其他很多网站

# AIStudio

# 如何进入 AI Studio

AI Studio 官网：https://aistudio.google.com/prompts/new_chat?pli=1

前提条件：你的节点必须是美国、新加坡、日本等其他符合条件的地区才能正常使用，如果是香港等不符合条件的地区，则会报错，然后会跳转到符合条件的地区列表


# 使用 AI Studio

# AI Studio API

# 获取 API

创建 API 密钥

选择工作区，点击创建

生成成功

# 使用 API

要使用 API，需要在对应的软件，或通过代码进行请求

软件推荐：Cherry Studio (樱桃工作室)

下载后，打开软件，点击设置

模型 ——> 找到 Google——> 填入 API 密钥

点击管理获取模型列表

添加你需要的模型

测试连通性

连接成功

回到聊天界面，选择聊天模型，就可以通过 API 进行聊天啦

# 通过国内代理 URL 进行聊天

前面提到，使用 AI Studio 需要美国、新加坡、日本等节点才能使用，API 也不例外，如果科学上网环境不允许，也可以通过代理 URL 的方式进行访问

https://api-proxy.me/gemini

# Gemini

AI Studio 是面向开发者的免费聊天模型官网，而 Gemini 是 Google 面向消费者的聊天官网

区别：一个是免费的，一个是收费的

前者的使用条件有更高的要求，后者只需要懂聊天即可
