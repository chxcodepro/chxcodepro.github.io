# 什么是域名

域名（Domain Name） 是互联网上用于标识和访问网站、服务器或其他网络服务的易记名称，说人话，方便记忆罢了

# 域名的作用

  • 代替 IP 地址：将难记的数字 IP（如 142.250.189.46 ）转换为易读的名称（如 baidu.com ）。
  • 品牌标识：如 baidu.com 、 taobao.com ，增强用户记忆和信任感。
  • 灵活管理：同一域名可绑定多个 IP 或服务器（如负载均衡）。

# 根域名，二级域名，三级以及其他域名

顶级域 .com （商业机构）、 .org （非营利）、 .cn （国家代码）， .edu （学校）， gov.cn （中国政府网站）等
一级域 127126 ，用户可自定义的部分（如企业名、品牌名）
子域名 www 、 mail 、 blog 等，用于区分同一域名下的不同服务（可选）

# 常见 DNS 参数

记录类型 用途
A 域名 → IPv4 地址
AAAA 域名 → IPv6 地址
NS 指定权威 DNS 服务器
CNAME 域名 → 另一个域名
MX（需要的时候了解即可） 邮件服务器地址
TXT 验证域名所有权

# SSL

HTTP+SSL 是用于加密网络通信的协议，确保数据在客户端（如浏览器）和服务器（如网站）之间传输时的机密性、完整性和身份认证。

# SSL/TLS 作用

加密传输，用户保护数据传输
