本教程基于博主 CM 喂饭，Youtube 教程搭建后，记录如何进行优选 IP，优化体验

workers 源码地址：https://github.com/cmliu/edgetunnel/blob/main/_worker.js

# 创建 KV 存储库

绑定名称 = KV

值 = KV存储库

# 创建密码

绑定名称 = PASSWORD

值 = 你的密码

# 优选 1（编辑优选列表）

优选 IP 地址

  1. https://stock.hostmonit.com/CloudFlareYes
  2. https://cf.090227.xyz/
  3. http://ip.flares.cloud/

优选域名（直接填入列表）

  1. visa.cn
  2. ip.sb
  3. time.is
  4. wto.org

指定端口： 443 2053 2083 2087 2096 8443

修改名称： #

# 优选 2（优选软件）

  1. https://github.com/XIU2/CloudflareSpeedTest

指定端口： ?port= 443 2053 2083 2087 2096 8443

修改名称： ?id=名称

# 优选 3（切换订阅器）

复制 TG 频道中优选订阅器的 sub=xxx.com

然后直接导入代理工具即可

# 优选 4（切换 PROXYIP）

复制 TG 频道中的 CMLiussss Push 的 URL

如要同时使用切换订阅器，加入 & 即可
